02 — From Zero

The quantum concepts, explained plainly

No prior quantum background assumed. Each idea here is exactly what the simulation below is built on — nothing extra, nothing skipped.

◯

Superposition — the spinning coin

A classical bit is 0 or 1. A qubit can hold both at once until it's measured — like a coin still spinning in the air, not yet landed on heads or tails. Only measurement forces a definite outcome.

|ψ⟩ = α|0⟩ + β|1⟩,   |α|² + |β|² = 1
⚯

Gates — H and CX

The Hadamard gate (H) puts a qubit into equal superposition. The Controlled-NOT gate (CX) flips a second qubit only if the first one is 1 — this is the operation that creates correlation between two qubits.

∞

Entanglement — the Bell state

Applying H then CX to two qubits produces (|00⟩ + |11⟩)/√2. Measure one qubit and the other's result is instantly fixed, no matter the distance between them — and neither qubit had a definite value beforehand. This is fundamentally stronger than any classical correlation.

𝜌

Density matrices — describing noisy states

A state vector only describes a perfectly clean ("pure") state. Once noise is involved, the qubit pair becomes a statistical mixture — this requires the more general density-matrix formalism (ρ), which is what powers the concurrence calculation below.

03 — The Proof

The CHSH / Bell test

This is the rigorous, textbook-standard way to prove entanglement rather than just claim it. Two qubits are measured at four angle combinations; the results are combined into a single score, S.

≤

Classical bound: |S| ≤ 2

If each qubit secretly had a fixed value all along (a classical "hidden variable"), Bell's theorem (1964) proves the combined score can never exceed 2 — under any measurement strategy.

√

Quantum limit: S = 2√2 ≈ 2.828

This is the Tsirelson bound — the maximum any quantum system can reach. Our clean, noiseless simulated Bell pair scored S = 2.8545, matching theory almost exactly.

CHSH value vs. channel noise

real simulation data
As depolarizing noise on the signal qubit rises, S falls smoothly — crossing below the classical bound of 2 around 12% noise. Full decay from 2.85 down to 0.42 at 50% noise.
04 — A Second, Independent Check

Concurrence: measuring entanglement directly

CHSH tells you whether entanglement beats the classical limit. Concurrence (Wootters, 1998) measures it directly on a 0–1 scale, computed from the actual noisy density matrix.

Concurrence vs. channel noise

real simulation data
Concurrence starts at a perfect 1.0 for a clean Bell pair and decays to 0.1466 at 50% depolarizing noise — an independent metric telling the same physical story as the CHSH curve above.
05 — The Application

Quantum illumination-style detection

Quantum illumination (Lloyd, 2008) is a real published concept: entangle two photons, send one toward a possible target through noise/clutter, keep the other at home, then jointly measure them. The surviving correlation helps detect the target even when the returning signal is buried in noise.

This simulation builds an honest discrete-variable stand-in: entangle two qubits, pass the "signal" qubit through amplitude damping (photon loss) and depolarizing noise (clutter), then use a Bell-basis measurement as an entanglement witness — compared against a fair classical baseline built from a genuine 20,000-trial Monte Carlo simulation under identical noise.

Why density matrices?

Noise turns a pure state into a mixed one. A state vector can't represent that — concurrence and fidelity both require the full density-matrix formalism, which is what this module uses throughout.

Why GPU-aware?

AerSimulator requests a GPU backend at runtime and falls back to CPU automatically if unavailable — detected live, never hardcoded, so the same script runs safely on any judge's machine.

08 — Scope, Honestly Stated

What this is — and isn't

Judges trust a team more, not less, when it's precise about the boundary of its own model.

Not a full continuous-variable QI system

Real quantum illumination uses squeezed light and homodyne detection. This is a discrete-variable qubit analogy inspired by the same principle — useful for demonstrating the effect, not a hardware-ready radar system.

Simulated, not run on physical hardware

All results come from Qiskit Aer's noise-model simulator. Real IBM Quantum hardware would add gate-calibration errors this model doesn't capture.

The classical baseline is genuinely fair

Built from real 20,000-trial Monte Carlo sampling under identical noise levels — not a fitted closed-form curve chosen to look worse than the quantum result.

Every number here is reproducible

Same script, same random seed (42), same simulator — anyone can re-run qt16_simulation.py and get these exact values back.
